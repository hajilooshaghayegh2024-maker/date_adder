# Date Adder

A simple Python CLI tool to add or subtract days, weeks, or months from a given date.

## Features
- Add/Subtract days, weeks, and months.
- Supports ISO 8601 format (YYYY-MM-DD).
- Defaults to today's date if none is provided.

## Usage
```bash
python date_adder.py --date 2023-01-01 --days 10
```
